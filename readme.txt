== Description ==

Change the pattern permalink of pages for best seo results, and delete the "category" prefix.

If page contains childs, its permalink pattern will be %pagename%/
If not, it will be %pagename%.html

If you haven't change the categories prefix, seo url pages delete the default value (category). Otherwise, it does nothing.

== Installation ==

1. Download 'seo-url-pages.zip'
2. unzip and upload 'seo-url-pages' directory to your '/wp-content/plugins' directory
3. Go to the plugin management page and enable the plugin
4. That's it !
